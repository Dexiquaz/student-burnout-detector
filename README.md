# Student Burnout Detection System

A non-clinical system designed to help students identify early signs of academic burnout
using study habits, sleep patterns, workload, and self-reported stress.

## Objective
To provide early warnings and actionable suggestions without offering medical advice.

## Status
Backend setup in progress.
